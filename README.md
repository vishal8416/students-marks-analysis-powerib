# 🎓 Student Performance Analysis Dashboard | Power BI Project

## 📌 Project Overview
This project is an interactive **Power BI Dashboard** created to analyze and visualize student academic performance using educational data insights.

The dashboard focuses on identifying how different factors such as:
- Gender
- Ethnicity
- Parental Education
- Test Preparation

impact student scores in:
- Math
- Reading
- Writing

The project transforms raw student data into meaningful visual insights using **Power BI, Power Query, and DAX**.

---

# 📊 Dashboard Features

## ✅ KPI Metrics
- Total Students
- Average Total Marks
- Pass Percentage
- Highest Score
- Lowest Score

---

## 📈 Analysis Included
- Student Performance Overview
- Subject-wise Score Analysis
- Gender-based Performance Insights
- Parent Education Impact Analysis
- Test Preparation Performance Analysis
- Performance Category Distribution
- Pass vs Fail Analysis
